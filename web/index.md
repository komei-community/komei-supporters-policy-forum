# 公明サポーターズ政策協議ポータル

このサイトは、GitHub リポジトリ「公明サポーターズ政策協議リポジトリ」で行われている政策提案・議論・採択状況を、GitHub アカウントを持たない方にもわかりやすく公開するためのポータルです。

## このプロジェクトについて

- 政策提案のオープン化
- 市民の声の集約
- 実現性の高い提案づくり

くわしい背景や運営方針は、GitHub 上の README をご覧ください。
