公明党の政策形成プロセスに、市民・サポーターがオープンに参加できるようにするための議論ポータルです。

新しい政策要望を投稿する（Googleフォームが開きます）

政策ポータルの概要

GitHub Issue / Pull Request を使って政策のアイデアを出し合い、議論し、ブラッシュアップした上で「採択済み政策」として公開していきます。

  • 提案・議論の履歴はすべて GitHub 上で公開
  • このサイトでは、その結果としてまとまった政策と、議論の入口（Issue）を一覧できます

主要メニュー

  • 提案中・採択済みの政策一覧を見る
  • 議題・ディスカッション（Issue）一覧を見る
