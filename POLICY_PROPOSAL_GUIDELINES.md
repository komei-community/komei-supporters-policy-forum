# GitHub Copilot Agent 用プルリクエストひな型作成プロンプト

## リポジトリ特性

このリポジトリは政党に政策を要望するための提案・検討リポジトリです。プログラムコードは含まれません。
政策提案のため一部法律案を提示する必要があります。


## 作成する記事の方向性

作成する記事は主に以下の通りです。
プルリクエスト作成時の本編ファイルは`draft-policy/`に配置します。Issue番号とタイトルをMarkdownのファイル名にします。
DeepResearch準拠のリサーチ時のエビデンス集は`source/`に配置します。

## 本編ファイル内容

### 前文案
法律の前段となる概要説明・背景説明を行います

### 条文案
実際の法律の条文を記載します。

### 現行の行政システムおよび生活実態
現行の仕組みや市民生活の実体をまとめます。これにはDeepResearch準拠の調査の結果も含みます。

### 現行の仕組みの問題点
現行の仕組みの問題点をまとめます。

### 本提案の条文によって解決されるケース
本提案の条文によって解決されるケースをまとめます。

### 本提案の条文に関する懸念事項
本提案の条文に関する懸念事項をまとめます。

## DeepResearch準拠調査結果
調査結果は一つのMarkdownにまとめ、各エビデンスをファイル内に列挙します。

### サマリー
600文字程度のまとめを記載します。
